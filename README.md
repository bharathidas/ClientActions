# Client Actions

This module provides a set of reusable JavaScript utility actions for handling user information, navigation, and application behavior in Mendix.

## Features:
### •	GetFileURL: 
Returns the accessible URL of a file stored in Mendix.
### •	GetToken_CSRF: 
Retrieves the CSRF token for secure client-server communication.
### •	GetUserId_GUID: 
Gets the GUID of the currently logged-in user.
### •	GetUserName:
Returns the username of the current user.
### •	GetUserRoleNames: 
Fetches all role names assigned to the current user.
### •	navigateBack:
Navigates one step back in browser history.
### •	reloadApplication: 
Reloads the entire application.
### •	UserIsGuest:
Checks whether the current user is a guest (anonymous).

## Dependencies:
Mendix Modeler 11.8.0 or newer.
